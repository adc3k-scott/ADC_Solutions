# Utility Grid Partnership Program (ADC-UTIL-001)

ADC's **open-source partnership offer to Louisiana electric
utilities** (decision
[0008](../../ops/decisions/0008-utility-grid-partnership-program.md)):
the 10 MW MGN-BOD-002 BTM block + dock-based drone inspection of the
utility's surrounding substations/feeders/ROW + the unified telemetry
plane as grid services + federal funding alignment
(GRIP/SPARK, 40101(d), USDA RUS, BRIC). The spec is public — the same
offer is visible to every utility, because the Entergy/SLEMCO/LUS
territories interlock around Lafayette. Drone ops model is
**vendor-neutral by design** — selected at each utility's working
session.
