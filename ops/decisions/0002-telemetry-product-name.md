# 0002 — The AI-native OS product is named "Telemetry"

- **Date:** 2026-06-10
- **Decided by:** Scott Tomsu, President
- **Status:** Accepted

## Decision

ADC Solutions' fifth product — the AI-native operating system layer
specified in ADC-OS-001 (`products/Telemetry/`) — is named **Telemetry**.
Plain "Telemetry", not "ADC_Telemetry" (the ADC-prefixed form was
explicitly rejected).

## Context

The platform layer (MGN-TEL-001 Layers 3–5, productized) needed a name
before the spec sheet, website, and commercial materials could carry it.
This was tracked as open item OS-01, which gated on the platform half of
MGN-BOD-002's COMM-001 ("company/product name → replaces MGN doc
prefix").

## Consequences

- OS-01 is resolved; `products/OS/` renamed to `products/Telemetry/`.
- COMM-001 remains half-open: the hardware-document prefix ("MGN") for
  the microgrid node docs is still undecided.
- Document IDs (ADC-OS-001, MGN-TEL-001) are unchanged — they are
  engineering identifiers, not the brand.
